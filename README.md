1. Project Overview

Cho-Han is a traditional Japanese dice gambling game where players bet on whether the sum of two dice is even (Cho) or odd (Han).

This project is a Python-based simulation of that game, supporting:

Manual betting (player-driven)
Automated betting (simulation over multiple rounds)
Data logging and persistence
Basic statistical tracking

The system is structured as a console-based application with supporting files for data storage and analysis.

2. Core Concept

At its heart, the program revolves around:

Rolling two dice
Determining parity (even/odd)
Comparing player bet vs result
Adjusting player balance accordingly

This simple mechanic is extended into:

Repeated simulations
File logging
Data analysis (via separate modules)
